```
보통 이들은 첫인상이 사근사근하고 친절하여 가까워지기 쉽지만 

어느덧 께름칙한 느낌과 함께 그 관계는 내가 그의 들러리로 이용 당한다는 소모감을 안겨준다.

...

내가 먼저 피해주는 것이.. 그들은 어쨌거나 자기 자신에게 밖에는 관심이 없으니까.
```







```
"괜찮아"라고 대답하는 대부분의 사람들, 사실 속으로는 하나도 괜찮지 않은 것이다.
```







```
목소리가 크고 공격적인 사람들을 피하십시오. 그들은 영혼을 괴롭힙니다.
```







```
밀물과 썰물 사이에서 어느덧 내 곁을 여전히 자연스레 지키고 있는 그 사람이 지금의 내 사랑스러운 벗이다.
```







```
나는 부탁을 함으로써 상대에게 감정 노동을 시킨 것이다.
```







```
"삶의 여유를 가져라"라고 주장하는 유명 인사가있다면,

사실 그분은 그 말을 하기 전까지 엄청나게 무리하고 노력하고 스스로를 혹사시킨 사람이었다는 거죠.
```







```
자발적으로 일을 만들어 하는 것을 끌어내리려는 사람들이 있잖아요.

그건 자기가 자기 무덤을 파는 거죠. 자기 열등감이고요.

보니까 자기는 그게 안 되거든요.

정말 그렇게 되고 싶지만, 그게 되지 않는 거예요. 시기죠.
```







```
열애랑 연애를 구분해요.

열애는 진짜 말 그대로 우리가 드라마에서 보는 뽀뽀하고, 설레고, 이런 거잖아요.

연애라는 게 오래가고 되게 부드럽거든요. 영원할 것 같은 그런 느낌 때문에요.
```







```
진짜 애도를 마치고 마음의 정리가되면 그다음부터는 좋은 기억만 남고

그 사람한테 고맙다는 생각이 들어요.

제 인생의 어느 한 시점에서 저를 행복하게 해준 것에 대해.

내가 그 사람을 행복하게 해줄 수 있게 허락해준 것에 대해.
```







```
한 사람 앞에서 나의 모든 것을 제대로 보여줄 수 있으면 인간관계에 관한 한 건강한 사람으로 봐요.
```

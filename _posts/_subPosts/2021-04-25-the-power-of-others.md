```
다음에 누군가 당신에게 "어디서 어떻게 지내세요?"라는 질문을 한다면,

좀 더 신중히 생각한 후 대답하자.

그 질문에 어떻게 답하느냐가 당신의 능력과 성장, 그리고 당신 삶에 대한 모든 것을 바꿀 수 있다.
```
 
 

 
```
당신의 나약함을 표출할 수 있는 곳이 아무 데도 없고,

에너지와 다른 자원들을 '다운로드' 할 수 있는 사람들의 네트워크에 연결될 수 있는 곳이 하나도 없다면,

무언가 정말 잘못된 것이다.

당신의 리더십, 수치, 두려움, 압박 혹은 습관 때문에 발전하지 못하고 실패할 수밖에 없는 한 코너로 몰린 것이다.
```
 



 
```
나쁜 연결이 반드시 폭력적이거나 약한 사람과의 연결을 의미하지는 않는다.

대신에 어떤식으로든 자신에게 부정적이고 '가치가 떨어지는'것 같은 느낌을 불러일으키는 사람과 연결되거나,

혹은 그런 사람에게 집착하고 끌리는 것을 말한다.

당신 자신이 열등한 것처럼, 또는 어떤 결함이나 문제가 있는 것처럼 느끼게 된다.

무슨 이유로든 이런 사람은 당신의 삶을 지배하고 당신에게 부정적인 감정을 심어 준다.
```
 



 
```
다른 사람의 존재 없이도 이런 자격지심을 느끼는 경우가 있다.
 
당신의 머릿속에 상주해 온, 내부의 심판관과 오랫동안 긴밀한 관계를 맺으면 이런 현상이 일어난다.

이 심판관은 당신의 성장기 혹은 과거에 당신에게 중요한 영향을 미쳤던 누군가의 잔재일 가능성이 높다.

그 사람에 의해 주입된 메시지가 당신의 칩에 늘 연결돼 있어,

스스로에게 부족하다는 반응을 남기고 스스로를 방어하게 만든다.
```
 



 
```
인간의 자기통제와 능력은 완전히 자기 자신의 통제력에 달려 있지만

그 자양분은 우리 능력을 형성하는 관계에서 대부분 제공된다.

그렇다, 과거든 현재든 늘 타인들은 우리가 자기통제력을 계발하는데 도움을 줬다.

이것이 능력의 역설이다.
```
 



 
```
좋은 관계는 각 이해 당사자가 서로에게 무엇을 기대하는지를 솔직하게 소통하는데 달려 있다.
```
 



 
```
서로 진실한 연결 상태를 유지하려면 어느 정도의 소통이 적당한지,

또는 너무 지나쳐서 상대방을 질식시켜 버리진 않는지 파악해야 한다.

상대방에게 기대하는 바를 명확히 밝히고 그 기대들을 실제로 이행하기 전부터 계속해서,

또는 필요하다면 간격을 두고 서로 연락하며 확인해야 한다.
```
 



 
```
뇌 회로에 대한 연구 결과를 보면, 어떤 문제를 바로잡고자 할 때 타인을 통해 그 방법을 배우거나

타인이 해결하는 광경을 지켜보는 것보다 자기 스스로 그 문제를 해결하려 애쓰는 편이 건설적이며,

이때 새로운 역량이 계발된다고 한다.

우리가 부정적인 감정을 경험하지 않을 때 더 오랫동안 집중하고, 더 명확히 사고하며,

정보를 더 잘 처리할 수 있다는 사실을 보여준다.
```
 



 
```
"가치를 무너뜨리는 사람들은 다른 곳으로 보내야 합니다.

그렇지 않으면 당신이 추구하는 것들을 그들이 파괴할 겁니다."
```
 



 
```
미하이 칙센트미하이가 설명했듯,

인간의 능력은 새롭게 얻은 기량이 더 향상되도록 과도하지 않은 수준에서 계속 도전할 때 그 정점을 찍는다.
```
 



 
```
인간의 물리적 욕구가 충족됐을지라도, 타인의 보살핌이나 사랑을 받지 않으면 다른 사람들을 사랑하고 유대감을 느끼는 능력을 계발할 수 없다.

마음이 공허한 사람은 타인을 사심없이 사랑할 수 없다.

그들은 불안과 욕구가 뒤섞인 진공상태 속에서 스스로를 작동해 나간다.
```
 



 
```
타인의 힘은 우리의 능력에 엄청난 영향을 미친다.

우리는 다른 사람들이 가져다주는 긍정적인 힘에 자신을 개방하지 않으면 다음 단계로 더 이상 나아갈 수 없다.

마음을 열고 수용하려면 자신의 나약함을 인정하고 욕구를 표현해야한다.
```
 



 
```
"당신이 직원들이나 고객들을 이해한다고 스스로 생각할 때는 그들을 진짜 이해하는 것이 아닙니다.

당신이 직원들이나 고객들을 이해한다는 사실을 '그들이' 이해할 때,

비로소 당신은 그들을 진정으로 이해한다고 말할 수 있어요."
```
 



 
```
우리는 상대방이 좋은 동기를 갖고 있고, 우리를 위해 좋은 일을 하고 싶어 한다는 사실을 알 때 그들을 신뢰한다.

사람들은 대개 중립적인 경우가 많다.

기본적으로 그들은 자신을 위해 조심하는 것뿐이다.
```
